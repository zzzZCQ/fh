# 企业微信自动化营销功能

## 功能说明

这个模块提供了企业微信网页版的自动化操作功能，可以模拟iPad浏览器登录，定时发送营销消息给客户。

**优点：**
- 不需要配置企业微信API
- 不需要配置可信IP
- 不需要购买域名
- 操作简单，只需扫码登录

## 安装依赖

```bash
pip install -r requirements_wecom_auto.txt
playwright install chromium
```

## 使用方法

### 1. 启动服务

```bash
python app.py
```

### 2. 访问管理页面

打开浏览器访问：`http://你的服务器:5000/admin/wecom-auto/`

### 3. 操作流程

#### 第一步：测试登录

1. 点击页面顶部的「测试登录」按钮
2. 会打开一个浏览器窗口
3. 使用企业微信扫码登录
4. 登录成功后会保持会话

#### 第二步：添加消息模板

1. 在「消息模板」标签页点击「添加模板」
2. 填写模板名称和消息内容
3. 保存模板

#### 第三步：创建定时任务

1. 在「定时任务」标签页点击「添加任务」
2. 填写任务名称
3. 粘贴客户列表（每行一个客户名称）
4. 选择要使用的消息模板
5. 设置发送时间
6. （可选）填写Cron表达式用于周期性任务
7. 保存任务

#### 第四步：查看发送记录

在「发送记录」标签页可以查看所有已发送的消息。

## Cron表达式说明

Cron表达式用于设置周期性任务，格式为：`分 时 日 月 周`

例如：
- `0 9 * * *` - 每天上午9点
- `0 9,18 * * *` - 每天上午9点和下午6点
- `0 9 * * 1` - 每周一上午9点
- `0 9 1 * *` - 每月1号上午9点

## 注意事项

1. 首次使用前请先点击「测试登录」进行扫码
2. 建议先用少数几个客户测试功能是否正常
3. 发送间隔默认3秒，避免发送过快被限制
4. 数据存储在 `wecom_data/` 目录下
5. 浏览器会话可能会过期，需要定期重新登录

## 技术架构

- **Playwright** - 浏览器自动化工具，用于模拟iPad操作企业微信网页版
- **APScheduler** - 定时任务调度器
- **Flask** - Web管理界面
- **JSON文件** - 数据持久化（模板、任务、发送记录）
